# Finax

A Finax é um sistema de gerenciamento de finanças pessoais que permite a separação de gastos por categoria, bem como a adição, remoção e edição de despesas e receitas. A plataforma oferece dashboards com resumos e insights financeiros em diferentes períodos, gerenciamento de cofres para objetivos e metas, suporte a múltiplos usuários, entre outras funcionalidades.

## Tecnologias ultilizadas

- HTML
- CSS
- JavaScript
- TypeScript
- React
- Tailwindcss
- Chart.JS
- Local e SessionStorage

## O que aprendi com esse projeto?

O desenvolvimento da Finax foi fundamental para consolidar e expandir meus conhecimentos em desenvolvimento front-end e arquitetura de aplicações. Ao longo do projeto, pude aprender e praticar diversos conceitos importantes, entre eles:

- Organização e arquitetura de aplicações React, separando responsabilidades entre componentes, layouts, páginas e serviços.

- Uso prático do TypeScript, criando tipos e interfaces para garantir maior segurança, legibilidade e previsibilidade do código.

- Gerenciamento de estado no front-end, lidando com dados dinâmicos como gastos, receitas, categorias e filtros por período.

- Persistência de dados no navegador, utilizando LocalStorage e SessionStorage para manter informações mesmo após recarregar a aplicação.

- Criação de dashboards e visualização de dados, aplicando o Chart.js para transformar dados financeiros em gráficos claros e informativos.

- Estilização moderna e responsiva com Tailwind CSS, acelerando o desenvolvimento de interfaces consistentes e reutilizáveis.

- Tratamento de regras de negócio, como cálculo de totais, filtros por data, categorias mais caras e metas financeiras.

- Planejamento de funcionalidades reais, simulando problemas do mundo real, como controle financeiro pessoal e múltiplos usuários.

- Esse projeto reforçou minha capacidade de transformar uma ideia em uma aplicação funcional, além de melhorar minha visão sobre boas práticas, escalabilidade e manutenção de código.
